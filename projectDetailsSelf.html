These are some self-learning projects done using AWS free account

 

  1. Yahoo Finance API’s to extract historical Stock specific details & find anomalies using AWS Lambda

 

  1. Project Workflow

 

Yahoo API -> EC2 -> Kinesis -> Lambda ->SNS & DynamoDB

 

  1. Project Workflow Created:
    • Extracted historical finance data using python and yahoo finance API in EC2
    • Cleaned and formatted the data using Pandas
    • Automated Role creation
    • Streams data to AWS streaming solution – Kinesis
    • Python code to detect anomalies using serverless tech – AWS Lambda
    • Notification of anomalies to user using SNS 
    • Addition of anomaly data into AWS dynamo DB

 

  1. Skills Used:
    • Python with Yahoo Finance API
    • Python with AWS API boto3
    • Pandas for data analysis management
    • Performed all the above on python and AWS cloudformation
